- 이름: 지훈
- 나이: 29세
- 직업: 서울 거주 직장인 (마케팅)

- 관심사: 감정 관리, 손으로 만드는 것, 혼자 있는 시간

🌐 [사용 시나리오: 주말 체험 추천]
1. 금요일 저녁, 지훈은 챗봇에게 말을 건넨다.
> “이번 주말에 뭐하지... 기분이 좀 가라앉는데 기분 전환하고 싶어.”

2. 챗봇은 감정 키워드 및 일정 정보 기반으로 대화를 이끈다.
> “지금 약간 지치신 것 같아요. 혼자만의 시간이 필요하신가요?”

3. 챗봇이 사용자 데이터를 참고해 매칭된 체험을 제안한다.
> “예산군의 목공 공방 체험은 어떠세요? 혼자 조용히 나무를 깎는 체험인데, 비슷한 성향의 사용자들이 만족도가 높았어요!”

4. 지훈은 "오, 재밌겠네" 하며 챗봇을 통해 예약까지 완료

5. 체험 이후, 챗봇은 다시 피드백을 요청한다.
> “체험 어떠셨어요? 기록을 남기시면 다음 추천에 반영할게요.”

6. 지훈은 만족스러운 후기를 남긴다.
> “혼자였지만 뭔가 차분해졌어요. 다음엔 가죽공예도 해보고 싶어요.”

7. 챗봇은 다음 체험을 예약할 수 있도록 관계를 지속적으로 이어간다.
> “그럼 다음 달엔 충북에서 진행되는 가죽공방 체험도 추천드릴게요!”

🔁 [유저 플로우 요약]
흥미 기반 대화 → 감정 분석 → 성향 맞춤 체험 제안 → 예약 → 체험 후 피드백 → 재추천 → 장기 참여 유도

📌 핵심 메시지:
- "흥미에서 시작된 경험이 관계로 이어집니다."
- 챗봇은 단순한 추천을 넘어서 감정, 성향, 행동 데이터를 기반으로 관계를 형성합니다.
