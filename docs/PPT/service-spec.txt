📄 서비스 명: 도시 밖, 지방 체험 큐레이션 챗봇

🧭 서비스 목적:
도시 청년들이 자신의 흥미와 감정을 기반으로, 지방에서 의미 있는 체험을 지속적으로 경험할 수 있도록 돕는 AI 챗봇 기반 큐레이션 플랫폼입니다.
기존 정부 주도의 단발성 지원 체험이 아닌, 사용자 중심의 ‘경험 흐름’을 설계하여 관계 인구로 확장되는 구조를 지향합니다.

🎯 핵심 문제 인식:
- 정부 정책은 대부분 단기적 인센티브나 금전적 유인에 머무름
- 청년은 금전보다 '경험', '취향', '다양성'을 중시
- 체험은 있으나, 나에게 맞는 체험을 고르기 어려움
- 지역 콘텐츠는 중복되고 진부화됨

🛠️ 주요 기능:
1. AI 챗봇 기반 대화형 인터페이스
   - 감정 기반 체험 추천
   - 일정, 지역, 관심사 맞춤 응답
   - 자연어 기반 유도 대화

2. 성향 기반 큐레이션
   - 감정/행동 패턴을 분석하여 개인 맞춤 체험 매칭
   - 예) “혼자 있고 싶은 날” → “조용한 목공 공방 추천”

3. 경험 흐름 설계
   - 체험 → 후속 피드백 → 재방문 추천 → 커뮤니티 초대 → 장기 체류 유도
   - 반복 참여 기반의 관계 인구 전환

4. 정책 데이터 피드백
   - 사용자 성향/패턴 데이터 수집
   - 정책 설계에 필요한 인사이트 제공

👤 주요 타겟 사용자:
- 20~40대 도시 거주자
- 반복된 일상에 지친 직장인 및 프리랜서
- 전공 외 체험, 공방/자연/로컬 기반 경험을 원하는 사용자

🧩 서비스 흐름 요약:
사용자 감정 입력 → AI 대화 → 체험 추천 → 예약 → 피드백 → 재추천

📊 기대 효과:
- 기존 체험 정책보다 지속성 높은 사용자 참여 유도
- 예산 낭비 최소화 및 인프라 재배치 전략 기반 제공
- 지역 콘텐츠와 도시인의 자연스러운 연결
- 스마트시티 정책 실패에 대한 실질적 대안 제공

🌍 장기 비전:
- 사용자의 체험 데이터를 기반으로 관계 인구 기반 지역 전략 수립
- 지속 가능한 스마트 커뮤니티 구성
- 일본 ‘관계 인구’ 모델의 국내 확장 버전 구현
